User Guide

Welcome to the DB Tools User Guide. This documentation is for Revit users who want to learn how to use DB Tools effectively.

What is DB Tools?

DB Tools is a collection of productivity tools for Autodesk Revit, designed to streamline structural engineering workflows. It provides automation for common tasks like:

  • Global Mapper (GM) - Batch parameter mapping between elements
  • Super Girt Tool (SGT) - Automated girt placement and management
  • Foundation Tags - Smart tagging for foundation elements
  • Transfer Drafting Views (TDV) - Import/export drafting views between projects
  • And many more...

Quick Start

  1. Install DBTools
  2. Open Revit and look for the DB Tools ribbon tab
  3. Select a tool and start working!
