DBTools.App

Overview

DBTools.App is the main application shell that hosts all tools and provides the Revit ribbon UI.

Location: csharp/src/DBTools.App/

Responsibilities

  • Revit ribbon tab creation
  • Tool button registration
  • Application lifecycle management
  • DI container setup

Key Components

Addin Entry

AddinEntry.cs - The Revit add-in entry point implementing IExternalApplication.

Ribbon Management

Creates and manages the DBTools ribbon tab with buttons for each registered tool.

Tool Hosting

Discovers and loads tools via DbtToolRegistry.

Output

After build and ILRepack, produces:

  • DBTools.App.{Year}.dll - Main add-in assembly
  • DBTools.addin - Revit manifest file
